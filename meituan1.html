项目背景

2020，注定是生鲜电商的一次大考。

因为年初突如其来的新冠疫情，线上购买生鲜果蔬成为了消费者的广泛选择。根据艾瑞数据于7月发布的《2020年中国生鲜电商行业研究报告》， 疫情使得大量用户涌入生鲜电商平台，市场交易规模显著提升。预计2020年交易规模接近4047.3亿元，相较去年增速为44.7%。

然而数据报告中也指出，特殊情况之下催生的新用户多为中老年群体， 而他们的日常买菜习惯很难因为短期的疫情影响而彻底改变。 因此，如何留存新用户，提高用户粘性以及活跃度成为了众多生鲜电商的待解难题。

在这个项目中，我的主要任务是为美团买菜的会员场景进行全流程交互设计，目的是提升用户对会员的吸引感知以及行动意向。 最终目标是通过鼓励会员制度提升用户的访问频次和购买频次。

在第一阶段的研究中，我设法找到如下三个问题的答案：

1. 我们的用户特征，线上买菜习惯以及对会员的观点是怎样的？
2. 现在美团买菜是如何吸引与维持会员的？
3. 美团买菜的竞品是如何吸引与维持会员的？

一、用户研究

我们的用户特征，线上买菜习惯以及对会员的观点是怎样的？

1.1 用户采访

我采访了25位使用线上购买生鲜服务的用户，其中10位来自我的熟人，15位为微博上随机调查。
我设计了一套针对用户线上买菜习惯，App选择，以及会员制度的采访问卷，目的是探索以下问题：

1. 用户使用线上买菜服务的原因/动力是什么？线上买菜的习惯怎样的？
2. 用户根据哪些方面选择App？
3. 用户对会员制是什么看法？如果注册了会员，考量因素和感知途径有哪些？

Figure 1: 用户采访问题提纲

1.2 亲和图法

我使用亲和图法整理了所有用户采访的回答，并且结合网络上数据报告进行了结果的分析。
我参考的主要数据来自中国生鲜电商市场年度综合分析2020, 《2020年中国生鲜电商行业研究报告》。

Figure 2: 根据用户回答，我制作了亲和图。

1. 用户使用线上买菜服务的原因/动力是什么？线上买菜的习惯怎样的？

  • 用户年龄段多集中在一二线城市80后，90后群体，有生活节奏快、收入高，熟悉互联网购物等特点。
    用户使用线上买菜的主要理由为：节省时间，时间段灵活，节省体力，没有去菜场习惯等。
  • 多数用户使用线上买菜服务频繁，每周约2-3次。
    大多数用户完全线上购物，少部分线上线下结合，采取线下购物的理由多为“需要亲自挑选”，“有线下买菜习惯”。

2. 用户根据哪些方面选择App？

  • 用户选择app的主要考量因素有：菜品新鲜程度，配送服务（速度，时间段）以及价格。
    其中，相对年轻的上班族非常重视灵活的配送时间与快捷的服务，而中老年用户更注重价格，菜品质量的挑选。 用户倾向于选择有口碑的平台（美团，京东，盒马）与自己有过线下体验的商店直送。
  • 用户对单一app忠诚度普遍较高。
    即使有使用多个app的用户也表示，自己会主要在其中一个app上购物。如此app不能提供某一品类，才会转向其他app。

3. 用户对会员制是什么看法？如果注册了会员，考量因素和感知途径有哪些？

  • 用户购买会员的主要理由为：价格实惠，信任，预计会长时间高频使用。
    已经注册的用户给出理由多为：自己预计以后经常会在该平台购物，计算发现会员制度下购物更加优惠。 没有注册的用户给出理由多为：自己不常用线上购物，没有详细了解过会员优惠。
  • 用户了解会员的主要方式为：首页，活动横幅，购物车页大幅宣传，朋友推荐。
    多数用户反馈通过app内的宣传了解到会员制度，包括首页，结账页面，个人主页，活动横幅等等。 部分app为吸引注册会员有返现活动或限时活动宣传。也有app推出邀请好友进行会员体验的活动。
1.3 用户分析与目标

根据调查结果，我将用户分为如下三类：普通用户-低频，普通用户-高频和会员用户。
为了使我的设计更加具有针对性，我针对每类用户的特征，制定了相应的设计目标（具体请见下图）。

Figure 3: 普通用户-低频，普通用户-高频和会员用户的特点以及相应目标。

目标

  • 普通用户-低频：通过新人活动，限时促销，优惠活动等提升其用户活跃度，培养线上买菜习惯，将其转化为高频用户。
  • 普通用户-高频：进行大幅会员宣传，会员注册相关活动，通过提升其对会员的感知，将其转化为会员。
  • 会员用户：通过提升会员使用体验（会员特权，定制服务等），提高用户粘性，鼓励其续订。
1.4 用户研究总结

在设计会员场景时，我的主要目标用户为普通用户-高频，会员用户。因为根据调查，使用频率较低的普通用户注册会员的意愿相对较低，因此我列为次要用户。
下一阶段我将以这两类用户为中心，对美团买菜现有产品进行深入研究。

针对前者，考察现有产品中会员宣传的曝光率，吸引度；
针对后者，考察现有产品中会员的用户体验。

二、现有产品研究

现在美团买菜是如何吸引与维持会员的？

我的第一步是深入考察美团买菜的用户界面，标注出引导用户进行会员注册的入口，并对比会员与非会员使用体验的异同。 我首先查看了菜单栏中导向的五个页面：主页，分类页，菜谱页，购物车页以及用户个人页，如下图。

我把可引导会员注册的部分分类为直接引导与间接引导：
直接引导——点击后直接导向会员页面。
间接引导——点击后导向会员有关活动，或标注会员福利，非直接导向会员页面。

Figure 4: 信息架构中五个最高层级的页面中会员引导。

2.1 主页：一个直接引导，一个间接引导。

“会员特权卡片” —— 导向美团买菜会员页；
“周五会员日” —— 导向会员活动页面，间接引导会员注册。

Figure 5: 主页中存在一个直接入口一个间接入口。

优点

  • "周五会员日"板块面积较大，保证了曝光度，有效吸引了用户注意力，实现引流效果。

缺点

  • "会员特权"板块面积较小，且在信息层级上与“领钱”“拼团”“夏日商品”等并级，容易被用户忽略。
  • “周五会员日”活动板块非常驻，例如8/22已被替换为七夕活动宣传，无法保证曝光一致性。
  • 两个入口整体出现位置较为靠后，非用户第一视觉聚焦区，需用户手动下滑。
2.2 分类页&购物车页：两个间接引导。

“会员专属”菜品 —— 特别分区：提高普通用户对会员的感知；提高会员的使用体验。
会员特价商品价格标注会员价与正常价 —— 提高用户对会员制度优惠力度的感知。

我尝试在普通用户与会员用户的两种情况下，将同一会员特价商品加入购物车。
(例：番茄——正常价3.4元，会员价3.1元。)

对比结果：
普通用户：将番茄加入购物车后，购物车页面番茄价格只显示正常价。
会员用户：将番茄加入购物车后，购物车页面番茄价格显示会员价并在旁边已划去正常价。

对比结论：购物车的会员价标签主要针对现有会员，提高会员用户对优惠力度的感知。

Figure 6: 分类页&购物车页：两个间接引导。

优点

  • 价格对比直观，便于用户感知会员优惠。

缺点

  • 对于单一菜品，会员与非会员价差距不大，用户无法强烈感知优惠力度。
  • 购物车页，普通用户不再看到会员价对比，无法查看会员对多种菜品的总体优惠。
  • 分类页，购物车页以及菜谱页均无可以一键跳转会员页面的直接入口，给用户了解会员造成不便。
2.3 个人主页：两个直接引导。

“开通会员卡”横幅 —— 直接导向美团买菜会员页；
“会员卡”功能区 —— 直接导向美团买菜会员页。

Figure 7: 个人主页：两个直接引导。

优点

  • 多个直接入口，出现位置仅次于头像ID，为用户第一视觉聚焦区。

缺点

  • 1. 用户与非用户个人主页没有区别设计。
    例如：已注册会员的用户看到的横幅仍为“开通会员卡”等吸引注册内容，缺乏针对性。
2.4 会员主页分析

会员主页包含：续费，会员特价商品，优惠券，以及规则。

Figure 8: 会员主页各区导向续费，会员特价商品，优惠券，和规则。

优点

  • 功能分区清晰，会员优惠一目了然。
  • 两个续费行动按钮，促进用户行动。

缺点

  • 用户与非用户会员主页没有区别设计。
    例如：已注册会员的用户看到的横幅仍为“开通会员卡”等吸引注册内容，缺乏针对性。
  • 会员特价商品区与优惠券区缺少行动按钮，例如“查看所有商品”，”领优惠券”。用户无法轻松判断下一步。
  • 缺少个人定制数据显示，例如“会员已为你节省xxx元”，“查看你的专属会员报告”。
  • 缺少互动环节，例如好友邀请活动，种树游戏等等，会员用户体验上有很大进步空间。
2.5 现有产品研究总结

在深入考察之后，我归结出了以下三个主要设计需求。

1. 首页，购物车页，分类页：需要进一步提高会员宣传曝光率。
2. 个人页，会员主页：需要会员和非会员区别设计，提高针对性。
3. 会员主页增加更多定制板块，增强用户体验。

针对以上三个需求，我对行业内同类产品开展了竞品分析， 对比了同类产品与美团买菜在吸引和保持用户方面的相似与异同。

三、竞品分析

美团买菜的竞品是如何吸引与维持会员的？

根据《2020年中国生鲜电商行业研究报告》， 中国生鲜电商行业商业模式可划分为多个品类。其中美团买菜所使用的前置仓模式，直接竞品为每日优鲜，叮咚买菜。 因此，我重点以每日优鲜和叮咚买菜这两款产品吸引用户的会员感知方面与美团买菜进行对比分析。

Figure 9: 《2020年中国生鲜电商行业研究报告》划分每日优鲜，叮咚买菜和美团买菜为前置仓模式。

3.1 首页，分类，购物车页：会员引导对比。

针对现有产品研究所产生的第一条设计需求“需要进一步提高会员宣传曝光率”，我对比了美团买菜，叮咚买菜和每日优鲜的首页，分类，购物车页的会员引导。

3.1.1 首页会员引导对比。

Figure 10: 首页会员引导对比。

可改进之处

  • 会员引导位置上移
    叮咚买菜以及其他电商基本将会员入口放在紧接搜索栏的第一部分内容，或者分类下内容，均属于用户聚焦区，引导效果更好。
  • 通过样式突出会员引导
    叮咚买菜的会员引导虽然占地面积不大，但样式非常突出，且没有相似组件，容易获得注意力。

Figure 11: 分类页会员引导对比。

优势

  • 会员分区明显
    相比叮咚买菜会员分区放在最末（需用户手动横拉），美团买菜的会员分区清晰易找。

可改进之处

  • 增加直接入口
    叮咚买菜在会员区设置了横幅，可直接跳转会员页面
3.1.3 购物车页会员引导对比。

Figure 12: 购物车页会员引导对比。

可改进之处

  • 增加会员引导
    叮咚买菜为非会员用户的购物车显示了会员可用的优惠券，在点击之后直接导向开通会员的入口，提高了会员转化率。 （注: 每日优鲜采取了相似的领券设计）
3.2 个人，会员主页：会员和非会员设计对比。
3.2.1 个人主页会员引导对比。

Figure 13: 个人主页会员引导对比。

可改进之处

  • 调整颜色+组件样式
    叮咚买菜通过使用黑白灰颜色，使得会员引导极其突出，吸引用户注意力。每日优鲜也采取其他内容全部粉色单色，凸显中间用户注册黑条。 相比之下，美团买菜所使用的高饱和度，明度，多种颜色使得会员横幅很难被用户第一眼注意到。会员卡的样式和领钱等活动并列，无法显出重要性。
3.2.2 会员和非会员显示内容对比。

Figure 14: 会员和非会员显示内容对比。

可改进之处

  • 针对会员应有内容和样式更改
    每日优鲜在用户注册之后会显示会员状态下已省xxx元。 相比之下，美团买菜除了将“立即开通”更换为”立即续费”之外，没有改动。
3.3 会员主页内容和服务对比

Figure 15: 会员主页内容和服务对比

优势

  • 美团买菜优惠标语最吸引用户（优惠额度最高），各分类可节省额度最高。
  • 续费入口清晰（两个行动按钮），并且价格具有优势。

可改进之处

  • 改进会员特权显示
    相比其他两块产品各列出6项和3项权益，美团买菜会员仅两项权益，但是实际上美团买菜会员也享有“周五会员日”等特权， 可以增加在此区域，提高吸引力。并且，叮咚买菜针对每项权益都设置了可点击的按钮，用户可被直接导航，美团买菜应在使用流畅度上做出改进。
  • 增加行动按钮
    会员特价商品区与优惠券区相比叮咚买菜缺少行动按钮，例如“查看所有商品”，”领优惠券”。
  • 增加升级/互动等环节
    相比每日优鲜的升级任务，共享会员等，缺少能显著增加用户参与感，互动率的环节。
3.4 竞品分析总结

在深入考察，对比竞品与美团买菜现有产品之后，我得出了以下设计总目标。（小目标可见前文详细分析）

1. 简化任务流程——在首页，分类，购物车页增加引导，用户可一键跳转注册会员。
2. 提高引导入口吸引力——通过样式，色彩等使其醒目，吸引用户聚焦。
3. 重新设计会员主页——定制会员专属页面内容，增加优惠品类。

明确设计目标以后，我开展了初步设计。

四、设计流程与思考

我重新设计首页，分类，购物车和个人页的主要目标是最大限度地简化用户注册会员的任务流程。

例如：
原来的产品中“用户在购物车页/分类页注意到会员优惠->切换到个人主页->从个人主页进入会员主页->注册”
可被简化为“用户在购物车页/分类页点击会员入口->注册”。
这样，我们可以使得用户注册会员的流程更加方便快捷，提升转化率。

4.5 重新设计会员主页以及功能

设计环节最大的挑战就是重新设计会员主页。 根据现有产品分析和竞品分析，会员主页需要针对以下设计目标进行重新设计：

  • 改进会员特权显示
    相比其他两块产品各列出6项和3项权益，美团买菜会员仅两项权益，但是实际上美团买菜会员也享有“周五会员日”等特权， 可以增加在此区域，提高吸引力。并且，叮咚买菜针对每项权益都设置了可点击的按钮，用户可被直接导航，美团买菜应在使用流畅度上做出改进。
  • 增加行动按钮
    会员特价商品区与优惠券区相比叮咚买菜缺少行动按钮，例如“查看所有商品”，”领优惠券”。
  • 增加升级/互动等环节
    相比每日优鲜的升级任务，共享会员等，缺少能显著增加用户参与感，互动率的环节。

改进会员特权显示

针对第一个问题，我将会员特权显示改变为横向的导航栏，用户可点击其中的图标直达相应板块。在功能上，我增加了会员专属报告，周五会员日的显示。
会员专属报告主要用于满足会员用户的定制化需求，增强用户体验，满意度。
周五会员日原本只可通过首页进入，我在新版本中增加了入口，便于用户一览。

增加行动按钮

针对第二个问题，我在会员特价商品区增加了行动按钮“查看400+商品”； 在神鲜券区增加了”立即领取”，更能促进用户作出行动，也增强了任务流程的流畅度。

增加升级/互动等环节

针对第三个问题，我在会员主页中增加了升级进度条，每升到新一级可解锁惊喜礼包。用户可以从进度条查看自己离下一级的距离， 并且可通过点击“做任务”直接被引导入活动区。这样可有效提升用户活跃率，促进购买频次。

五、设计成果

我使用Figma制作了可交互的简单原型，展示了各页面至会员主页的引导入口和会员主页的设计。

会员引导概览

我在首页，分类，购物车和个人主页添加了通向会员主页的直接引导。

相比现有的版本，我做了如下改动：
1. 首页升高了入口位置，改变了样式。
2. 分类页增加了横幅直接入口。
3. 购物车增加了横幅直接入口。
4. 个人主页改变了入口样式。

会员主页概览

会员主页包含：基本信息，标语，升级进度，会员特权导航，专属报告区，特价商品区，神鲜券区，运输券区，周五会员日入口。

相比现有的版本，我做了如下改动：

2. 增加了升级，定制服务等环节
3. 增加了特权种类与导航栏。

六、总结与反思

我们的下一步是什么？

6.1 如何测试设计成果？

我们可以通过数据搜集以及研究用户眼动来测试设计结果。
数据搜集将围绕用户的会员转化率，会员用户的续订率，活跃度来进行结果的比较，如果新的设计方案带来了会员转化率的提升，则视为有效的方案。
眼动实验主要围绕会员引导的模块是否能快速，成功吸引用户注意力，并进行分析。如果新的设计方案使得用户花费更短时间完成注册会员的任务，并且更快地注意到会员引导模块，则视为有效的方案。

6.2 设计上的不足

我认为首先在用户主页的服务品类上还有许多可以挖掘的内容，因为时间限制，我没有来得及仔细研究。
例如，会员升级制度按照什么标准升级，如何更进一步促进用户消费；会员报告具体包括哪些内容，用户最关心的数据是哪些？ 今后如果有机会，我会更进一步挖掘以上问题，优化我的设计。

6.3 个人感悟

这次的设计挑战锻炼了我搜集陌生数据的能力与独立研究的能力。 因为我身在美国，我对国内的线上生鲜app一开始一无所知。因此，我从零开始进行亲身体验， 搜集国内的数据报告并且注册了美团买菜的会员进行不同场景的研究，和国内的陌生用户进行远程的采访，非常有趣！
我非常享受分析一个已经投入市场，并且和居民生活密切相关的产品。 很感谢得到这次宝贵的锻炼机会！
